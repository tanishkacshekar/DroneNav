## 📖 Project Overview

DroneNav is an autonomous drone navigation system developed to enable intelligent and efficient drone movement in dynamic environments. The project combines path planning, real-time obstacle detection, and autonomous navigation to ensure drones can safely reach their destinations with minimal human intervention.

The system continuously processes sensor data, analyzes surrounding obstacles, and dynamically updates flight paths, making it suitable for applications such as aerial surveillance, disaster response, smart agriculture, logistics, and autonomous inspection.
